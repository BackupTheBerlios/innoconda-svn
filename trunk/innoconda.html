Innoconda

A new way of creating Windows installers with Inno Setup.

Motivation / Overview

I created Innoconda to automate the excellent Inno Setup with Python. It provides a very simple Python API for generating Inno Setup Scripts and invoking the Inno Compiler, ISCC.exe, to turn them into packages. Creating a .iss file, even with a wizard, can be a tedious process and very few things change. Innoconda lets you customize just what’s important, and build an installer with very few steps.

There is also an included PythonScript class. This has special support for makers of Python packages who don’t want to use distutils for some reason. Currently, it only works with “pure” Python packages (those without extensions written in C), but there are plans to support more complex Python packages.

How-To

Basic Installer (any application)

Use the Script class for any standalone application, even applications written in Python (these are sometimes called applications with private Python modules). See below for packaging libraries written in Python.

  1. Install Innoconda using the installer (see Downloads). You do not need to install Inno Setup, as it is included in Innoconda.

  2. Make sure you have win32all installed. You can download win32all from the Starship site.

  3. In the Python interpreter, try the following:

>>> import inno
>>> scr = inno.Script(display_name = "My Application", name = "myapplication", version = "1.0")
>>> scr.collect("myapplication") # find all the files in myapplication for packaging
>>> scr.compile() # invoke iscc

Python Installer

Use the PythonScript class for packages that are intended to be imported by other applications—that is, libraries, written in Python, and installed in site-packages. Your documentation and uninstaller will go into the default destination, "{pf}\". The end user will choose which version of Python to install the package into (if there is a choice), and everything else will go into whichever site-packages directory corresponds to the end user’s choice.

>>> scr = inno.PythonScript(display_name = "My Python Library", name = "mypythonlib", version = "1.0")
>>> scr.collect("mypythonlib")
>>> scr.compile()
